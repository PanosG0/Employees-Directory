CRUD Spring Boot Project

This project is a simple CRUD (Create, Read, Update, Delete) application using Java Spring Boot, Thymeleaf for the frontend, and MySQL for data storage.

Description

The application allows users to manage a basic dataset through a web interface. Users can:

- Create new entries
- View all entries
- Edit existing entries
- Delete entries

The application uses Thymeleaf for rendering views and Spring Data JPA for database interaction with MySQL.

Technologies
- Java 17+
- Spring Boot 3+
- Spring Data JPA
- Thymeleaf
- MySQL
- Maven
